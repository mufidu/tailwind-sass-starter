# Tailwind and Sass Starter Template

A starter template for using Tailwind and Sass.

## Installation

Use the package manager [npm](https://www.npmjs.com) to use this.

```bash
npm install
```

## Running locally

```bash
npm run watch
```

## Building for production

```bash
npm run build
```
